Welcome to the HKDSE ICT Interactive Learning App focusing on Recursion and the Call Stack, critical concepts in the Algorithm and Programming elective/core module.

## How to use:

Language Toggle: Switch between Traditional Chinese and English using the button in the header.

Code Preview: Switch between Python and C++ implementations to see how syntax differs while the logic remains the same.

Visualization: Use the "Input n" field to set a number (1-5 recommended), then click "Next Step" or use Right Arrow Key to step through the recursion.

Trace Table: Observe how variables change in each call.

Quiz: Test your understanding at the bottom of the page.

## Learning Tips:

Focus on the Base Case. Without it, the function calls itself infinitely, leading to a Stack Overflow.

Watch the Call Stack carefully; notice how it grows during "winding" and shrinks during "unwinding" after the base case is reached.
